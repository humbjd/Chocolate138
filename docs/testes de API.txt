TESTES DE API

O que é uma API?

Interface de comunicação e processamento entre funções e sistemas.

Exemplo do Ifood

App Ifood ----> Identifica ----> Lista  -----> Exibir o cardápio --------> Pagamento -------> Confirmação ----->
          Abre - Voçe / GPS - Informa de Restaurantes
          API                   API             API                         API                 API
Rastrear o pedido  ------> Receber
                           Reclamar

É a forma de dois ou mais sistemas conversarem

Operações de uma API:

Banco de Dados (BD)
Orientação a Objetos (OO)
                                                                    Em API
(C) Create                      Incluir                             Post
(R) Reach/ Read / Research      Consultar                           Get
(U) Update                      Atualizar                           Put (completa) / Patch (parcial)
(D) Delete                      Excluir (Lógica/Física)             Delete

LGPD - Lei Geral de Proteção de Dados ----> Direito de ser esquecido
